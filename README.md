# Dev-Eagle
![Logo](https://github.com/hishamdalal/Dev-Eagle/blob/master/src/logo.png?raw=true)

Web Development Starter via Vite

### How to use:
1. download or clone repository
2. Install dependencies
  ```
  npm i
  ```
3. Run for development or production 
  ```
  npm run dev
  npm run build
  ```
